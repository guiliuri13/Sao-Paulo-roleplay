Твоя супер безопасность не так уж велика, дядя! Легко.

Place vespucci in resources, then add to your server.cfg:
start vespucci

Enjoy!


Want lifts with lift doors?
Copy files from directory 'extra/doors' to 'stream' directory.

Want to bring back open lifts?
Copy files from directory 'extra/no_doors' to 'stream' directory.

Crashing with error message about PoolSize?
Remove all MLO's in neighbourhood, FiveM config portal amount limitation!
